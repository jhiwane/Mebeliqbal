Panduan Integrasi GitHub:

Karena website ini dihosting statis di Vercel, Anda tidak bisa mengupload file langsung dari halaman ini. Gunakan form di bawah untuk membuat Kode JSON produk. Copy kode tersebut, lalu paste ke dalam file data.json di GitHub Anda. Jangan lupa upload foto secara manual ke folder /assets/ di GitHub.

Buat Data Produk Baru

Pastikan nama ini sama persis dengan nama foto yang Anda upload ke folder /assets/ di GitHub.

2. Langkah Selanjutnya:

  • Buka folder assets di GitHub. Klik Add file -> Upload files.
  • Upload file foto Anda dan pastikan namanya persis: .
